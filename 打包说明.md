# 货物排版计算工具 - 打包说明

## 快速开始

由于当前环境存在文件锁定问题，请按以下步骤手动打包：

### 方法一：使用批处理脚本（推荐）

1. 关闭所有可能访问 `f:\Lingma` 文件夹的程序
2. 双击运行 `打包.bat` 文件
3. 等待完成后，在 `dist` 文件夹中找到 `货物排版计算工具.exe`

### 方法二：手动命令

```cmd
cd f:\Lingma

# 清理旧文件（如果存在）
rmdir /s /q node_modules
del /q package-lock.json

# 安装依赖
npm install electron electron-builder --save-dev

# 打包
npm run build
```

### 方法三：使用已创建的快捷方式

如果以上方法都失败，可以使用 `启动货物排版工具.vbs`：
- 双击运行即可在浏览器中打开应用程序
- 这不是真正的exe，但可以在Windows上快速启动

## 输出文件

打包完成后，exe文件位于：
- `f:\Lingma\dist\货物排版计算工具.exe`
